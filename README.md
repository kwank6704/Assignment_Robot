# Wix Hotel Booking Automation

## Project Overview
This repository contains an automated end-to-end testing suite for the Panaryco Hotel booking platform (https://panaryco.wixsite.com/myhotel). The automation is built using Robot Framework and SeleniumLibrary, designed to validate form submissions and handle complex UI components such as custom listboxes and asynchronous elements commonly found on Wix-based websites.

## Technical Specifications
- **Automation Framework**: Robot Framework
- **Driver Engine**: SeleniumLibrary
- **Target Browser**: Google Chrome
- **Environment Management**: Global Python Environment

## System Requirements
To run this project, ensure that Python 3.x is installed globally on your system. The following libraries must be installed:

pip install robotframework robotframework-seleniumlibrary
